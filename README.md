# phaser_first_game

## description
This is a basic phaser3 game. It utilizes a basic phaser structure, adds assets and defines their behavior.

Steps to play
~~~
git clone git@github.com:crowBottom/phaser_first_game.git
~~~

open the command line and and control shift v

cd into the game directory
~~~
cd phaser*
~~~

create a tempory server on port 8000
~~~
python3 -m http.server
~~~

in the url bar add the following
~~~
http://127.0.0.1:8000/part10.html
~~~

Use the arrow keys to play. Enjoy!

Oh yeah, and watch out for bouncing bombs...
